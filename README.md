# Cudex

[Cudex](https://hex.pm/packages/cudex) is a CUDA library in Elixir. Here is an example:

```elixir
iex> 

```

See the [online documentation](https://hexdocs.pm/cudex).

## Installation

Add to your ```mix.exs``` file:

```elixir
def deps do
  [
    { :cudex, "~> 0.0.1" }
  ]
end
```
